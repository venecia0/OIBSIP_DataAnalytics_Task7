# Credit Card Fraud Detection using Machine Learning

## Project Overview

This project focuses on identifying fraudulent credit card transactions using machine learning techniques. Detecting fraud is an important task in the financial sector because it helps organizations reduce financial losses and improve transaction security.

## Dataset

**creditcard.csv**

The dataset includes details of credit card transactions. It contains several anonymized features, the transaction amount, and a target variable that indicates whether a transaction is legitimate or fraudulent.

## Objective

The main objective of this project is to develop a machine learning model that can distinguish between normal and fraudulent transactions based on the given transaction data.

## Steps Performed

The following steps were completed during the analysis:

1. Loaded and explored the dataset
2. Examined the distribution of fraudulent and non-fraudulent transactions
3. Prepared the data and applied feature scaling
4. Split the dataset into training and testing sets
5. Trained a machine learning model using **Logistic Regression**
6. Evaluated the model using various performance metrics
7. Visualized the results using charts and evaluation matrices

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

# Outcome

The trained model was able to identify fraudulent transactions with good accuracy. The evaluation metrics and visualizations helped assess the effectiveness of the fraud detection model.

## Conclusion

This project demonstrates how machine learning techniques can be used to detect fraudulent financial transactions. It highlights the importance of data preprocessing, model training, and evaluation in building reliable fraud detection systems.
